# github-app

### Quick start

```bash
# clone the repo
$ git clone https://github.com/lapusneanugabi/github-app.git github-app

# change directory to github-app
$ cd github-app

# install the dependencies with npm
$ npm install

# start the server
$ npm start
```

go to [http://localhost:8080](http://localhost:8080) in your browser.
